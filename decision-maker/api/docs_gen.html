Associate devices to a workflow

Associate new devices to a workflow. The request body should contain all devices which should be part of the workflow matching the given id. You may also use this endpoint for removing devices from the workflow, by simply ommiting them from the request body. That is to say, if a workflow contains devices 1, 2, and 3, by using this endpoint with only devices 1 and 3, the workflow will now contain only devices 1 and 3.

Authorizations:
accessToken
path Parameters
id
required
string

Workflow ID

Request Body schema: application/json
required
devices
required
Array of strings

An array of device IDs to associate with this workflow

Responses

/api/workflow/{id}/devices

Request samples

  • Payload
Content type
